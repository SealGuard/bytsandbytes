# Byts & Bytes — Corporate Site

Clean, premium marketing site for **Byts & Bytes LLC**.

> We don't build software. We engineer futures.

## Design

- **White** background with subtle cyan accents
- Centered looping logo video hero (`Untitled design.mp4`)
- Soft circuit canvas, paper portfolio, services, contact

## Stack

- Tailwind CSS (CDN)
- Custom CSS + canvas circuits
- Vanilla JS (portfolio, video autoplay)

## Run

```bash
cd byts-and-bytes
python3 -m http.server 4173
```

Open [http://localhost:4173](http://localhost:4173).

## Hero video

`assets/logo-anim.mp4` — autoplay, loop, muted, centered, large but not full-screen.
